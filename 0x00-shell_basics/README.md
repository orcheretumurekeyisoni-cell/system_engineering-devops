# 0x00-shell_basics

This directory contains the Shell Basics project for the System Engineering and DevOps course.

The following scripts are included:

1. **0-current_working_directory** – Prints the absolute path of the current working directory.
2. **1-listit** – Lists the contents of the current directory.
3. **2-bring_me_home** – Changes the current directory to the user’s home directory.
4. **3-listfiles** – Lists the contents of the current directory in long format.
5. **4-listmorefiles** – Lists all contents, including hidden files, in long format.

Each script starts with `#!/bin/bash` and is executable.
